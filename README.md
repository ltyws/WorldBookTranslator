# 世界书翻译器

一个基于Java和JavaFX的图形界面世界书翻译工具，支持将英文世界书翻译为中文。

## 功能特点

- 图形界面，操作简单，适合小白用户
- 支持自定义API地址、密钥和模型
- 支持缓存功能，断网重连后可继续翻译
- 实时显示翻译进度和日志
- 支持查看和编辑世界书条目
- 支持自定义界面标题和背景颜色
- 自动保存翻译进度，每处理5个条目保存一次缓存

## 系统要求

- Java 8 或更高版本
- Windows操作系统

## 安装和使用

1. 确保已安装Java 8或更高版本
2. 双击运行 `start.bat` 文件启动应用程序
3. 在界面中填写API配置信息：
   - API地址：例如 https://api.openai.com/v1/chat/completions
   - API密钥：您的OpenAI API密钥
   - 模型：例如 gpt-3.5-turbo
4. 点击"选择JSON文件"按钮，选择要翻译的世界书文件
5. 点击"开始翻译"按钮开始翻译
6. 等待翻译完成，翻译完成后会自动生成输出文件

## 功能说明

### 主界面
- **API配置**：填写API地址、密钥和模型
- **文件选择**：选择要翻译的JSON文件
- **开始翻译**：开始翻译过程
- **查看条目**：查看和编辑世界书条目
- **设置**：自定义界面标题和背景颜色
- **进度显示**：显示翻译进度和日志

### 条目查看界面
- **条目列表**：左侧显示所有条目
- **条目详情**：右侧显示选中条目的详细信息
- **保存修改**：保存对条目的修改

### 设置界面
- **标题**：自定义应用程序标题
- **背景颜色**：自定义界面背景颜色（例如：#f0f0f0）

## 注意事项

- 翻译过程中请勿关闭应用程序
- 如果翻译过程中断网，重新启动应用程序后会自动从缓存恢复进度
- 翻译完成后，输出文件会保存在原文件同目录下，文件名为"output_" + 原文件名
- 缓存文件会在翻译完成后自动清除

## 常见问题

**Q: 如何获取OpenAI API密钥？**
A: 访问 https://platform.openai.com/ 注册账号并获取API密钥。

**Q: 翻译速度慢怎么办？**
A: 可以尝试更换更快的模型，或者检查网络连接。

**Q: 翻译出错怎么办？**
A: 查看日志区域的错误信息，检查API配置是否正确，或者网络连接是否正常。

**Q: 如何修改翻译错误？**
A: 点击"查看条目"按钮，选择要修改的条目，修改后点击"保存修改"按钮。
